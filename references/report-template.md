# Report Template

Use this structure for first-pass reports:

1. Data overview
   - Source query or URL
   - Number of notes
   - Total and average engagement

2. Top notes
   - Title
   - Author
   - Likes, collects, comments
   - URL

3. Frequent tags and terms
   - Repeated tags
   - Repeated title/body terms

4. Reusable content angles
   - Top-note content summaries
   - Candidate title structures
   - Follow-up topic ideas

5. Token-platform insight section
   - Add this only when token-platform credentials are configured.

6. Comment insights
   - Add when `comments.normalized.json` is non-empty
   - Frequent comment terms
   - Top liked comments
   - User questions and pain points
   - Candidate follow-up topics
